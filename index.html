Git Clone:

Steps:

  1. First, create a Global Repository
  2. Copy The URL
  3. Type Git clone url in command line
  4. create one index.html and edit something
  5. do check status and Commit
  6. Now push to global
  7. While push if u get err msg like fatal current branch and orgial branch not matching
  8. then type "git branch --unset-upstream"
  9. and then type "git push"
  10. and git will tell you "fatal: The current branch newbranch has no upstream branch. To push the current branch and set the remote as upstream"
  11. and then type "git push --set-upstream origin branchname
